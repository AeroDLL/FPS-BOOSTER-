## ⚡ Features (22+ Tools)

### 🚀 Performance & Gaming
- **Ultimate FPS Boost:** Activates the "Ultimate Performance" power plan.
- **GPU Turbo Mode:** Enables Hardware-Accelerated GPU Scheduling & High Priority.
- **BCD Tweaks:** Disables dynamic ticks and platform clock for raw performance.
- **Gaming Runtimes:** Auto-installer for essential gaming libraries.
- **RAM Cleaner:** Clears the "Working Set" memory to free up RAM.

### 🌐 Network & Connectivity
- **DNS Optimizer:** Switch between Cloudflare (1.1.1.1), Google (8.8.8.8), or Auto.
- **Network Repair:** Resets TCP/IP stack and Winsock to fix lag/packet loss.
- **Ping Test:** Built-in latency benchmark tool.
- **Throttling Fix:** Disables network throttling in the registry.

### 🛠️ System Maintenance
- **Repair Station:** Runs `SFC` and `DISM` to fix corrupted Windows files.
- **Software Updater:** Updates all installed applications via `winget`.
- **Startup Manager:** View and manage startup applications.
- **Deep Clean:** Removes Temp, Prefetch, DNS Cache, and Browser Cache.

### 🛡️ Privacy & Security
- **Privacy Shield:** Disables Windows telemetry, tracking, and advertising ID.
- **Defender Control:** Temporarily disable Real-Time Protection for gaming.
- **Secure Backup:** Creates a System Restore Point before applying tweaks.

## 📦 Installation & Usage

### Method 1: Download EXE (Recommended)
No Python installation required.
1. Go to the **[Releases](../../releases)** page.
2. Download `AeroFPS.exe`.
3. Right-click and select **"Run as Administrator"**.
4. Select your language and enjoy!

### Method 2: Run form Source (For Developers)
```bash
# Clone the repository
git clone [https://github.com/AeroDLL/FPS-BOOSTER-.git](https://github.com/AeroDLL/FPS-BOOSTER-.git)

# Install dependencies
